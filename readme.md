# Estudos em Python

Este repositório reúne meus estudos iniciais em Python, incluindo exercícios práticos, anotações e pequenos programas desenvolvidos para treinar lógica de programação.

## Conteúdo

- Exercícios básicos

- Estruturas condicionais (if, elif, else)

- Operações matemáticas e tipos de dados

- Prática com entrada e saída de dados

- Scripts simples para fixação

## Objetivo

Concentrar meu progresso em Python em um único lugar, mantendo organização e facilitando futuras revisões.

---

## Como executar

Certifique-se de ter o Python instalado na sua máquina.

Execute os arquivos pelo terminal usando:
```
python nome_do_arquivo.py
```
*Alguns exercícios utilizam input(), então é necessário digitar valores durante a execução.*
