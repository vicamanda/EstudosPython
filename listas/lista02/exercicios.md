# Lista de Exercícios - Laços de Repetição em Python 

**Nível Básico** 

---

1. Contagem de 1 a 10 

Escreva um programa que imprima os números de 1 a 10 usando um laço for.

---

2. Números pares de 0 a 20 

Crie um programa que imprima todos os números pares de 0 a 20. 

---

3. Tabuada de um número 

Desenvolva um programa que mostre a tabuada de um número específico (por 
exemplo, 5) de 1 a 10. 

---

4. Soma dos primeiros N números

Faça um programa que calcule a soma dos primeiros 10 números naturais (1 + 2 + 
3 + ... + 10). 

---

5. Contagem regressiva 

Crie um programa que faça uma contagem regressiva de 10 a 1 e depois imprima 
"Fogo!". 

---

**Nível Intermediário** 

---

6. Fatorial de um número 

Escreva um programa que calcule o fatorial de um número (por exemplo, 5!).

---

7. Verificar se um número é primo 

Desenvolva um programa que verifique se um número é primo. 

---

8. Sequência de Fibonacci 

Crie um programa que gere os primeiros 10 números da sequência de Fibonacci. 

---

9. Padrão de asteriscos 

Faça um programa que crie um triângulo de asteriscos com 5 linhas.

---

10. Números perfeitos 

Desenvolva um programa que encontre todos os números perfeitos até 1000. 

---

**Nível Avançado (com Listas)**

---

11. Soma de elementos de uma lista 

Escreva um programa que calcule a soma de todos os elementos de uma lista de 
números. 

---

12. Encontrar o maior elemento 

Crie um programa que encontre o maior número em uma lista. 

---

13. Filtrar números pares 

Desenvolva um programa que crie uma nova lista contendo apenas os números 
pares de uma lista original. 

---

14. Inverter uma lista 

Faça um programa que inverta os elementos de uma lista sem usar métodos built
in. 

---

15. Contar ocorrências 

Escreva um programa que conte quantas vezes cada elemento aparece em uma 
lista.
